Clustering with K-Means¶

K-means is one of the simplest clustering algorithms, which you perhaps are already familiar from Data Mining. The K-means Algorithm consists in writing a program that proceeds as follows:

  1. Choose value for K (number of clusters);
  2. Initialize the locations of the centroids, which we can do by randomly select K points from your dataset as initial locations of the centroids;
  3. Calculate distance of all other points to each of the K centroids;
  4. Associate each point to the cluster of the closest centroid;
  5. Update the centroid position, by computing the average coordinates of all points associated to each cluster;
  6. Evaluate the average change in the centroids positions, as a measure of convergence (the algorithm is said to have converged to the solution when the positions of the centroids don't change more than a given tolerance threshold);
  7. Repeat steps 3-6 until either the centroids no longer move more than a tolerance threshold or until you repeated these steps at least for specified number of iterations (niter)

The algorithm is thus parametized by the number of clusters (K), the maximum number of iterations (niter), and the tolerance threshold (tol).

def init(points, K):
    # first we initialize a Random Generator, that will allow us to generate random numbers using many different methods
    rng = np.random.default_rng()
    # and then, using choice(), we generate a random sample from the given array
    # we use replace = False because we want a sample without replacement
    return rng.choice(points, K, replace = False)

def cupdate(points, clusters):
    # first we create a pandas dataframe with the points, where the first column corresponds to the first coordinate and the second column corresponds to the second coordinate of the points
    information = pd.DataFrame(points, columns = ['1st coordinate', '2nd coordinate'])
    # then we add a column to the dataframe, called cluster, with the cluster corresponding to that point
    information['cluster'] = clusters
    # finally we use the groupby function in order to calculate the average of the coordinates in the points for each cluster
    information = information.groupby('cluster').mean()
    # then we return the pandas dataframe as a numpy array to easier mathematical manipulation
    return information.to_numpy()

def pupdate(points, centroids):
    # number of points
    d = points.shape[0]
    # number of centroids
    k = centroids.shape[0]
    
    # ndarray that stores the distance between each of the d points and the k centroids
    distance_matrix = np.zeros((d, k))
    # 1-dimensional array that stores the index of the centroid that is closer to each point
    updated_clusters = np.zeros((d), dtype = np.int8)
    for i in range(d):
        for j in range(k):
            # filling in the distance matrix using the distance function created earlier
            distance_matrix[i][j] = distance(points[i], centroids[j])
        # the point with index i is associated to the index of the centroid (in other words, the cluster) closest to that point
        updated_clusters[i] = np.where(distance_matrix[i] == distance_matrix[i].min())[0][0]
    return updated_clusters

def error(oldcentroids, newcentroids):
    # first we compute a matrix where each row represents the distance between the two points of the same row in the two matrices (oldcentroids and newcentroids)
    distances = np.sqrt(np.sum(np.square(oldcentroids - newcentroids), axis = 1))
    # then we calculate the average of all the distances
    return np.mean(distances)

def kmeans(points, niter, tol, K = 2):  # setting the default value of K to 2
    
    # correct range of values for K
    # the number of clusters cannot be bigger than the number of total points, and also cannot be negative or zero
    if (K > points.shape[0]) | (K <= 0):
        return None
    
    # initializing the positions of the K centroids using the function defined above init()
    old_centroids = init(points, K)
    # setting the number of iterations initially to 0
    num_iterations = 0
    
    # we want to repeat the following steps until we reach a specified number of iterations (niter)
    while num_iterations <= niter:
        # 1-dimensional array that contains the clusters associated to each point, by computing the minimum distance
        new_clusters = pupdate(points, old_centroids)
        # numpy ndarray that contains the positions of the centroids associated to each cluster
        new_centroids = cupdate(points, new_clusters)
        
        # if the average euclidian distance between the old and the new centroids is below the tolerance level, we stop the algorithm
        if error(old_centroids, new_centroids) < tol:
            break
        else:
            # update the old centroids to the new centroids
            old_centroids = new_centroids
            # update the number of iterations
            num_iterations += 1
        
    return new_clusters, new_centroids

def inertia(points, clusters, centroids):
    number_clusters = np.unique(clusters).size
    # first we create a numpy matrix with the positions of the centroids associated to each point
    all_centroids = np.take(centroids, clusters, axis = 0)
    # then we calculate the squared euclidean distance of the points in the two matrices (the one with the coordinates of the points and the one with the coordinates of the centroids)
    squared_distances = np.sum(np.square(points - all_centroids), axis = 1)
    # lastly we calculate the average of the squared distances
    return number_clusters, squared_distances.mean()
